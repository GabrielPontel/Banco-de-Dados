# Banco de Dados

Repositório contendo resoluções de exercícios de atividades, listas e provas, realizadas no decorrer das disciplinas des Banco de Dados do curso de bacharelado em ciência da computação do IFSP - Câmpus de Presidente Epitácio, realizados por mim, discente Gabriel Pontel de Mori, no ano de 2025.

## _Conteúdo Abordado_:
- Modelagem
- DDL (Data Definition Language)
- Consulta
- Funções de manipulação de Strings
- Trigger
- Function e Procedure
- View
- Criação de usuários e seu gerenciamento
